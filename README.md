# push_swap

このプロジェクトは、与えられた整数のスタックを、限られた操作セットを用いて昇順にソートするアルゴリズムを実装している。

## ファイル構成と説明

### `Makefile`
- プログラムのコンパイル、クリーンアップを行うためのルールを記述。`make`で`push_swap`が、`make bonus`で`checker`が生成される。

### `main.c`
- `push_swap`プログラムのエントリーポイント。引数を解釈し、スタックを初期化し、ソートアルゴリズムを実行する。

### `main_bonus.c`
- `checker`プログラム（ボーナス課題）のエントリーポイント。標準入力から受け取った命令をスタックに適用し、ソートされているかを確認する。

### `includes/`
- **`push_swap.h`**: `push_swap`プログラムのヘッダーファイル。構造体（`t_list`, `t_ops`, `t_best_move`）、enum（`t_op_type`）、および関数のプロトタイプ宣言を含む。

### `includes_bonus/`
- **`push_swap_bonus.h`**: `checker`プログラムのヘッダーファイル。`push_swap.h`をインクルードし、ボーナス課題特有の関数のプロトタイプ宣言を含む。

### `libft/`
- `push_swap`プロジェクトで利用される、標準Cライブラリ関数を独自に実装したライブラリ。

### `srcs/algo/`
- ソートアルゴリズムの中核をなすロジックを含む。
- **`sort.c`**: スタックのサイズに応じて、`small_stack_sort`または`main_stack_sort`のどちらのソートアルゴリズムを使用するかを決定する。
- **`small_stack_sort.c`**: 要素数が2, 3, 4, 5の場合のスタックをソートするアルゴリズムを実装。
- **`main_stack_sort.c`**: より大きなスタックに対する主要なソートアルゴリズムを実装。スタックAからスタックBへ要素を移動させ、その後戻すための最適な操作を計算する。
- **`calc_cost_set.c`**: 様々な回転操作の組み合わせに対する総コストを計算する。
- **`calc_cost_1.c`, `calc_cost_2.c`**: 個々の回転操作（`ra`, `rb`, `rra`, `rrb`, `rr`, `rrr`）のコストを計算するヘルパー関数。
- **`set_ops.c`**: 最適な操作に必要な各回転操作の回数を設定する。
- **`calc_and_set_ops.c`**: 最小コストの操作セットを決定し、操作を設定するための適切な関数を呼び出す。
- **`find_node.c`**: スタック内の最大値または最小値を持つノードを見つけたり、ノードの位置を取得したりする関数を提供。

### `srcs/checker/`
- `checker`プログラムのロジックを含む。
- **`apply_bonus.c`**: 標準入力から読み取った命令をスタックに適用する。
- **`get_next_line_bonus.c`**: ファイルディスクリプタから一度に1行ずつ読み取る関数で、命令を取得するために使用される。
- **`print_result_bonus.c`**: スタックが正しくソートされていれば "OK" を、そうでなければ "KO" を出力する。

### `srcs/common/`
- `push_swap`と`checker`で共通して使用されるユーティリティ関数を含む。
- **`parse_args.c`**: コマンドライン引数を解析してスタックを構築する。
- **`check_error.c`**: 入力エラー（数字以外の文字、整数オーバーフロー、重複）をチェックする。
- **`init.c`**: スタックやその他のデータ構造を初期化する。
- **`free.c`**: スタックやその他の構造体に割り当てられたメモリを解放する。
- **`limits.c`**: 数値文字列が整数の範囲内にあるかを確認する。
- **`max_min.c`**: 2つの整数のうち大きい方または小さい方を返す。
- **`print_error.c`**: 標準的なエラーメッセージを出力する。
- **`utils.c`**: 文字列配列の複製、配列サイズのカウント、文字列配列から整数配列への変換、スタックがソート済みか空かのチェックなど、その他のユーティリティ関数。

### `srcs/stack/`
- スタック操作のための関数を含む。
- **`list.c`**: 双方向連結リストの基本的な操作関数（`new`, `add_front`, `add_back`, `size`, `last`）。
- **`operations_1.c`**: `swap`と`push`操作、およびそれらの操作名を出力するバージョンを実装。
- **`operations_2.c`**: `rotate`と`reverse_rotate`操作、およびそれらの操作名を出力するバージョンを実装。
- **`repeat_rotate.c`**: 回転操作を指定された回数だけ繰り返す。
